AIS provides a complete turnkey Blade Inspection System consisting of Data Acquisition Hardware, Scanner, Scan Controller, Advanced Data Analysis Software, Couplant Tank, Transducers, Bubblers and Wedges. Advantages of the AIS Blade Inspection System are many fold:

  1. One company provides the entire system
  2. Turn key integrated data system with scanner
  3. High Resolution, Band Pass Filtering, Wave Form Averaging
  4. Inspect Blades of any length
  5. Motorized RoboScanner
  6. Report generating analysis software
  7. AIS provides support and / or an upgrade path for the life of its products
